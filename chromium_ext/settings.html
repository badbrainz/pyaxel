Before you begin using pyaxelws, be sure to download the pyaxelws Python scripts. You'll also need Python (2.x series) installed. Once these requirements have been fulfilled, simply open a command prompt and run the pyaxelws.py script.

python pyaxelws.py

The default port number that the server will listen on is 8002. If you recieve an error stating that the address is already in use, then try another port. Use the "p" option to set the port. For example:

python pyaxelws.py -p 8003

To see a full list of configuration options, use the "h" option:

python pyaxelws.py -h

After the port setting has been configured properly, you're ready to test the connection using the web browser. From the settings page, which you're viewing now, enter the port number and then press the Test Connection button.

Version

The original pyaxel is a CLI-based download accelerator in Python that works seamlessly in networks that are behind proxy servers and for protocols like http and ftp. Features present in pyaxel 0.1 are accelerated downloads, persistent reconnection, resumable downloads, download speed limiting, and download progress indication.

Pyaxelws is a pyaxel clone that introduces new features such as HTML5 Websocket server implementation, a Javascript library that provides an interface to the server, as well as a client application designed for the Chromium web browser.
