Before using pyaxelws, download the pyaxelws server files. You'll also need Python (2.x series) installed. From a command prompt, run the server.py script.

python server.py

The default port number that the server will listen on is 8002. If an error states the address is already in use, try another port. Use the "p" option to set the port. For example:

python server.py -p 8003

To see a full list of configuration options, use the "h" option:

python server.py -h

Read the pyaxelws wiki to learn more advanced methods of starting the server. After the server has been configured properly, test the connection using Google Chrome. From the settings tab, press the Test Connection button.

Version

The original pyaxel is a CLI-based download accelerator in Python that works seamlessly in networks that are behind proxy servers and for protocols like http and ftp. Features present in pyaxel 0.1 are accelerated downloads, persistent reconnection, resumable downloads, download speed limiting, and download progress indication.

Pyaxelws is a pyaxel clone that introduces new features such as an HTML5 Websocket server, and a client application built for the Chromium web browser.
